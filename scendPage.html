Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: gets the HTML markup with tags and comments.
innerText: gets only the test content on the element.
textContent: It also gets the text content on the element. But it does not consider CSS styles

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element instead of adding individual event listeners to each child element

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event that is triggered on a child element propagates through the DOM tree to its parent elements, all the way to the root (usually the document object).

Question-5: How do you create, add, and remove elements using JavaScript?

Create elements: document.createElement(tag name);
Add elements: parentElement.appendChild(createElement);
remove elements: parentElement.removeChild(childElement);
